Vocabulary Learning Program KOREAN DOWN BELOW
Introduction
This program is a simple tool designed to assist with vocabulary learning. It loads words and their definitions from a data file, allowing you to study them. Incorrectly answered words are automatically added to a "Wrong Notes" list, which can be saved as a separate file for later review. After saving the wrong notes, the program will automatically close.

Key Features
Data File Selection :
Select a txt file in the "word - definition" format to begin learning.
Learning Progress :
Study a randomized list of words in sequence.
Type each word and check if your answer is correct.
Incorrect answers are automatically added to the "Wrong Notes."
Wrong Notes Management :
View a table of incorrectly answered words and their definitions.
Save the "Wrong Notes" as a txt file for future review.
The program will automatically close after saving the wrong notes.
User-Friendly Design :
A clean layout with pastel colors provides a calm and focused learning environment.
Installation and Execution
Executable File :
The program has been distributed as an .exe file.
Locate the VocabularyLearning.exe file in the downloaded folder.
Running the Program :
Double-click the VocabularyLearning.exe file to run the program.
No additional installation or setup is required.
Preparing the Data File
Format :
The data file must be in txt format, with each line written in the "word - definition" format.

apple - A fruit that is red or green.
banana - A long yellow fruit.
cat - A small domesticated carnivorous mammal.
File Location :
The data file can be placed in the same folder as the program or selected via a custom path.
How to Use
1. Main Screen
Select Data File :
Choose a data file containing the words and definitions you want to study.
Start Learning :
Begin studying using the selected data file.
View Wrong Notes :
Check the list of incorrectly answered words and their definitions.
2. Learning Screen
Word Display :
A word will appear on the screen. Type the word into the input field.
Answer Verification :
Press the Enter key to check your answer.
Correct: A "Correct!" message will appear along with the definition.
Incorrect: An "Incorrect." message will appear, and the word will be added to the "Wrong Notes."
Move to the Next Word :
Click the "Next" button or press the Enter key to move to the next word.
Return to Main Screen :
Click the "Return to Main Screen" button to go back to the main menu.
3. Wrong Notes Screen
View Wrong Notes :
Incorrectly answered words and their definitions are displayed in a table format.
Save Wrong Notes :
Click the "Save Wrong Notes" button to save the notes as a txt file.
The saved file will follow the "word - definition" format.
The program will automatically close after saving the wrong notes.
Return to Main Screen :
If you choose not to save the wrong notes, click the "Return to Main Screen" button to return to the main menu.
Saving Wrong Notes
Choose Save Location :
Click the "Save Wrong Notes" button to open the file-saving dialog.
Specify the desired location and file name.
File Format :
The saved file will be in txt format, structured as follows:

apple - A fruit that is red or green.
banana - A long yellow fruit.
cat - A small domesticated carnivorous mammal.
Automatic Closure :
After saving the wrong notes, the program will automatically close.
Notes
Data File Format :
The data file must strictly follow the "word - definition" format.
Incorrect formatting may prevent the program from reading the data properly.
Using the Enter Key :
You may need to press the Enter key twice to move to the next word in some cases.
